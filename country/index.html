Country

Each rectangle represents a country (code is in the rectangle) Size of the rectangle is proportional to the total number of prefixes by ASN registered in that country (IPv4) Color of the rectangle is proportional to the percentage of deaggregated prefixes relative to the total this country originates (green = low, via yellow to, red = high) Hover will show a tooltip with country code, number of IPv4 addresses and deagg percentage
